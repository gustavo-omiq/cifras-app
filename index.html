Proxies CORS (download das cifras)

O navegador não pode acessar o cifraclub.com.br diretamente (bloqueio CORS), então o download passa por um destes proxies, testados em ordem. Um por linha; {url} = endereço codificado, {raw} = endereço puro.

Proxy próprio (recomendado — Cloudflare Worker grátis)

Os proxies públicos podem ficar instáveis. Para ter um proxy só seu, crie um Worker gratuito em workers.cloudflare.com com o código abaixo e adicione https://SEU-WORKER.workers.dev/?url={url} na primeira linha da lista acima.

export default {
  async fetch(req) {
    const url = new URL(req.url).searchParams.get("url");
    if (!url || !/^https:\/\/(www\.)?cifraclub\.com\.br\//.test(url))
      return new Response("URL inválida", { status: 400 });
    const r = await fetch(url, { headers: { "User-Agent": req.headers.get("User-Agent") || "Mozilla/5.0" } });
    const h = new Headers(r.headers);
    h.set("Access-Control-Allow-Origin", "*");
    h.delete("Content-Security-Policy");
    return new Response(r.body, { status: r.status, headers: h });
  }
};

Como usar no iPad

1. Hospede estes arquivos em qualquer site estático (GitHub Pages, Netlify…) e abra no Safari — depois toque em Compartilhar → Adicionar à Tela de Início. O app passa a abrir e funcionar totalmente offline (as cifras ficam salvas no próprio aparelho).

2. Alternativa sem hospedagem: copie o index.html para um app leitor de HTML que suporte JavaScript e armazenamento (ex.: Koder, Documents). A biblioteca fica salva enquanto o app mantiver os dados do site.

Em ambos os casos, faça backups periódicos pelo botão acima.
